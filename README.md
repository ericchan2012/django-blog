1，至少能每3天进行一次代码提交直到项目完成

3，以功能模块的形式进行项目推进，能认真负责的完成各个功能
4，以github形式进行代码共享
5，需要提交的代码发给我，由我来merge并提交到github上
6，主动沟通交流项目进行中的问题
